アプリ制作

いくつかの質問に答えるとその条件にあったアニメを絞っておすすめしてくれるアプリです。
AIに「アニメに詳しい」というロールを書き加えることでより幅広いアニメを進めてくれるようにしました。

ソースコードはこちら↓
  1. from openai import OpenAI
  3. #APIを使うkeyを入力します。これは絶対にwebページなどで公開してはいけません。
  4. import os
  5. client = OpenAI(api_key="")
  7. #以下の質問から条件を絞っていく
  8. a=input("今の気分を入力")
  9. b=input("今なりたい気分")
  10. c=input("今見たいジャンルを入力")
  11. d=input("最近ハマっていること、ものを入力")
  12. e=input("興味のあること、ものを入力")
  13. f=input("どのくらいの時間で見終えたいか（何分、何クール）入力")
  14. g=input("一番アニメに求めている要素を入力")
  17. question = a+b+c+d+e+f+g+"に合うアニメを教えて"
  19. #ここがAPIです----------------------------
  20. response = client.chat.completions.create(
  21.     model="gpt-4o-mini",
  22.     messages=[ {"role": "system", "content": "あなたはアニメに詳しいユーザー嗜好分析AIです。"},
  23.         {"role": "user", "content": question}]
  24. )
  25. #---------------------------------------
  27. print("AIの答え：", response.choices[0].message.content)
 制作理由
自分の欲しいものが何か考えた時によく次に見るアニメを何にしようか迷ってしまうため、このシステムは嬉しいなと思ったからです。
